Form submit

Work with form submit

Requirements: When you submit the form (on submit event), page should't reload. When you submit the form, in console should situated data from all fields (in object) all field should be required (by js, not by html), if any field is empty, form should not be submitted

Advice: add event listener to form, not to button. use event.preventDefault() to prevent page reload use trim method on string to remove whitespaces before and after string (for more info check mdn)
